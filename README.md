# 正念 - uTools 插件

一款极简禅意风格的正念冥想辅助工具，帮助你在忙碌的工作中随时停下来，回归当下。

## ✨ 功能特性

### 🍃 呼吸练习
- **4-7-8 呼吸法**：吸气 4 秒、屏息 7 秒、呼气 8 秒，快速平复焦虑
- **箱式呼吸法**：吸-屏-呼-屏各 4 秒，帮助专注与恢复精力
- 双层圆环实时动画引导，跟随圆圈的缩放节奏调整呼吸

### 🧘 冥想计时
- 支持 3 / 5 / 10 / 15 / 20 / 30 分钟自由设定
- SVG 圆环进度条可视化倒计时
- 开始 / 暂停 / 重置 完整控制
- 冥想完成自动计入统计并打卡，播放轻柔提示音

### 🫧 放空
- 100 秒倒计时，放空大脑，等待灵感浮现
- 过程中若想到要做的事情，可随时点击「想到了」结束
- 倒计时结束自动播放提醒音

### 🌿 心情记录与打卡
- 5 种心情状态（开心 / 平静 / 低落 / 焦虑 / 疲惫）一键记录
- 自动统计连续打卡天数与累计冥想时长
- 打卡日历一目了然，本月坚持情况清晰可见

### 📖 每日正念语录
- 精选 15 条来自中外哲人、禅师、经典典籍的正念语录
- 每天自动轮换，为你的片刻休息带来一句启发

## 🎨 设计风格

纯黑底色 + 暖金强调色 + 大面积留白，极简禅意风
缓动微动画、细线圆角卡片，营造安静、专注的氛围

## 📦 安装方式

1. 打开 uTools
2. 搜索「正念」「冥想」「呼吸」「静心」任一关键词
3. 回车即可进入插件

或在 uTools 插件市场搜索「正念」安装。

### 技术栈

- 原生 HTML / CSS / JavaScript（无框架依赖）
- Web Audio API 生成提示音
- uTools dbStorage / localStorage 数据持久化

## 💾 数据说明

所有数据（心情记录、冥想时长、语录索引）均通过 uTools 本地存储持久化，无需联网，保护隐私。

存储的 Key：
- `mindfulness_moods` - 每日心情记录
- `mindfulness_total_seconds` - 累计冥想秒数
- `mindfulness_quote_date` - 今日语录日期标记
- `mindfulness_quote_index` - 语录索引
